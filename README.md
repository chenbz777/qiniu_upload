## 介绍

七牛云文件上传

<br />

## 快速开始

#### 安装依赖

```bash
# 推荐cnpm
npm install
```

#### 运行

```bash
npm run dev

# open http://localhost:7001
```

<br />

## 接口说明

#### 接口

```bash
/upload
```

#### 类型

`post`

#### 入参

| 参数 | 是否必填 | 描述 |
| ---- | -------- | ---- |
| file | ✅        | 文件 |
